# Calibrate a PerpetualBooster model

Calibrate a PerpetualBooster model

## Usage

``` r
perpetual_calibrate(model, x, y, x_cal, y_cal, alpha)
```

## Arguments

- model:

  A `PerpetualBooster` object.

- x:

  Validation features.

- y:

  Validation targets.

- x_cal:

  Calibration features.

- y_cal:

  Calibration targets.

- alpha:

  Calibration parameter.
